#DESCRIPTION
* Example using Semaphores to solve Dining Philosopher problem.
* Each philosopher has a chopstick.
* In order to eat, they need to have a pair chopsticks.
* They can pick it from their neighbors.

#QUICK-START:

* Compile the project:
$ make build

* Run Java application:
$ java Main $1 $2

with $1 is the number of philosophers,
     $2 is the number of eating rounds.
